# Visualizador de Perfil do GitHub

## Descrição

Este é um projeto de um visualizador de perfil do GitHub que permite aos usuários pesquisar por um nome de usuário do GitHub e visualizar informações do perfil, incluindo:

-   Avatar
-   Nome
-   Bio
-   Número de seguidores
-   Número de pessoas que o usuário segue
-   Uma lista dos 10 repositórios mais recentes do usuário

## Como Executar

1.  Clone este repositório para sua máquina local.
2.  Abra o arquivo `index.html` em seu navegador de preferência.

## Como Usar

1.  Digite o nome de usuário do GitHub que você deseja pesquisar no campo de busca.
2.  Clique no botão "Buscar" ou pressione a tecla "Enter".
3.  As informações do perfil do usuário serão exibidas na tela.

## Tecnologias Utilizadas

-   HTML
-   CSS
-   JavaScript

## Funcionalidades

-   Busca de usuário do GitHub em tempo real.
-   Exibição de informações do perfil do usuário.
-   Exibição dos 10 repositórios mais recentes do usuário.
-   Design responsivo.
